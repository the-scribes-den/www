## 🚀 Getting Started

### Clone the Repository

```bash
# Clone the repository
git clone https://github.com/the-scribes-den/www.git

# Navigate to the project directory
cd www
```

### Run Locally

Open the `index.html` file in your preferred browser to view the site locally:

```bash
# On macOS/Linux
open index.html

# On Windows
start index.html
```
